# Сценарий демо-медиа

Что снять для портфолио. Перед записью: однотонные обои в чате, крупный шрифт,
чистое окно. Гифку сжать рецептом из корневого CLAUDE.md (цель < 10 МБ).

## Подготовка
1. Запустить бота с настроенным `GIGACHAT_CREDENTIALS` (живые ответы выглядят лучше).
2. Иметь второй аккаунт-«владелец», куда падают заявки и эскалации (OWNER_CHAT_ID).

## Скринкаст `demo.gif` (один дубль)
1. `/start` → показать приветствие и меню (3 кнопки).
2. Вопрос по базе: «сколько стоит стрижка?» → бот отвечает «1500 ₽».
3. Вопрос по базе: «вы работаете в воскресенье?» → бот: воскресенье выходной.
4. Вопрос ВНЕ базы: «делаете маникюр?» → бот честно отшивает + кнопка «Оставить заявку».
   (Параллельно у владельца в чате появляется «❓ Вопрос без ответа…».)
5. Жмём «Записаться» → имя → контакт → удобное время (или «Пропустить») → «Отправить».
6. Показать у владельца уведомление «🆕 Новая заявка» и строку в `leads.xlsx`.
